# 🌱 GrowTracker

GrowTracker is an AI-powered pantry management application that helps users track groceries, monitor expiry dates, reduce food waste, and manage kitchen inventory efficiently.

## ✨ Features
- 📦 Pantry Management
- 📷 Grocery Scanner
- ⏰ Expiry Date Alerts
- 🤖 AI Recipe Suggestions
- 🛒 Smart Shopping List
- 📊 Pantry Dashboard

## 🛠️ Tech Stack
- React.js
- Node.js
- Express.js
- Supabase
- Tailwind CSS
